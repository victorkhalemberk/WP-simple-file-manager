== Description ==
WP Simple File Manager is a powerful and feature packed file management plugin, allowing you to manipulate files and directories in your website file system.


== Features ==

1. Bulk select

2. Bulk delete

3. Bulk copy

4. BulK move

5. File download

5. KeyBoard shortcuts

6. File and folder compression

7. Archive extraction
